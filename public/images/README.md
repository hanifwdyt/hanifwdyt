# Images Directory

Store images for your content here.

## Structure

```
images/
├── writing/       # Images for blog posts
│   └── [slug]/   # Organize by post slug (optional)
└── projects/      # Images for projects
    └── [slug]/   # Organize by project slug (optional)
```

## Usage in Markdown

```markdown
![Alt text description](/images/writing/your-image.jpg)
```

## Supported Formats

- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif`

## Tips

- Use descriptive file names
- Optimize images before uploading
- Recommended max width: 1200px
- Use WebP format for better performance
